# Extraction Template

Use this for each raw source.

```markdown
## Source
- Type:
- Path/URL:
- Date/context:
- Confidence: high / medium / low

## One-line Thesis

## Mechanism
What actually creates the result? Avoid vague claims.

## Evidence / Metrics
List metrics, examples, workflows, or concrete artifacts.

## Constraints
What must be true for this idea to work?

## Transferable Pattern
What can be reused in another project?

## Non-transferable Parts
What is specific to this source/context?

## Project Implications
- Keep:
- Improve:
- Delete/defer:
- New asset candidate:
```
