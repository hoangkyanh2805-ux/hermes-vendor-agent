## 9. BOT DEPLOYMENT SEQUENCE — Cài Đặt Bot Vào Group

### Thứ tự deploy bot

| # | Hành động | Ở đâu | Nội dung |
|---|----------|-------|---------|
| 1 | **Thêm bot vào group** | Telegram group | Add @hermes7979_bot làm admin group |
| 2 | **Set bot permissions** | Group settings | Read messages, send messages, delete messages |
| 3 | **Tạo vendor-bot/.env** | VPS /root/hermes-bin/vendor-bot/ | Copy config từ section 8.5 |
| 4 | **Deploy vendor_listener** | VPS | Script listener quét group + route agent |
| 5 | **Deploy vendor_nudge cron** | VPS crontab | `0 1 * * * /root/hermes-bin/vendor-bot/nudge.sh` |
| 6 | **Deploy vendor_reminder cron** | VPS crontab | `0 11 * * * /root/hermes-bin/vendor-bot/reminder.sh` |
| 7 | **Deploy weekly_report cron** | VPS crontab | `0 2 * * 1 /root/hermes-bin/vendor-bot/weekly.sh` |
| 8 | **Test bot reply** | Vendor group | Gõ /help → bot phải trả lời list command |
| 9 | **Test nudge** | Vendor group | Chạy thủ công nudge → check message đúng |
| 10 | **Dry run 1 ngày** | Vendor group | Bot chạy 24h → check log → fix nếu cần |
