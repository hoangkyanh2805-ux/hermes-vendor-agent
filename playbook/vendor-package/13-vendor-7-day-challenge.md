## KHÁC BIỆT: 21-Day (cũ) vs 7-Day (mới)

| Tiêu chí | 21-Day (cũ) | 7-Day (mới) |
|----------|------------|------------|
| Độ dài | 21 ngày | 7 ngày (+ auto D14) |
| Mỗi ngày | 1 challenge dài | 1 action cụ thể, 15-30ph |
| Target | Người mới hoàn toàn | Dân affiliate (đã hiểu model) |
| Focus | Học từ từ | Làm được NGAY |
| Check-in | Tự tick tracker | Auto (bot D0/D1/D3/D7) + Manual (admin D5) |
| Follow-up | Không có | Segmented D14 (4 nhóm) |
| Commission | Flat 10K | Tiered (clone Accesstrade) |
| Dashboard | Không có | EPC/CVR/Approval Rate (clone Accesstrade) |
| Tool link | Không có | UTM + Link rút gọn (clone Accesstrade) |
| Leaderboard | Không có | EXP + streak + top tuần (clone The Allin Plan) |
| Nộp bài | Gửi group | Nộp → Review → Approve/Redo (clone The Allin Plan) |
| Nguồn | Tự nghĩ | Clone Accesstrade + The Allin Plan + UpPromote |
```

```

4. LÊN RANK: Tích lũy joins → Active → Top → Senior
5. GIÚP NGƯỜI MỚI: Khi bạn là Top/Senior, mentor vendor New trong group
6. CHECK LEADERBOARD: Xem thứ hạng — cạnh tranh lành mạnh

Tháng đầu: mục tiêu 3-5 joins.
Tháng 2-3: scale lên 20+ joins/tháng → lên Top.
```
