## DAY 0 — Nhận Công Cụ + Đọc Luật

> **Khi nào:** Ngay khi bạn join group Telegram
> **Thời gian:** 10 phút
> **Mục tiêu:** Có source tag + tracking link trong tay. Hiểu luật chơi.

### Công cụ cần hôm nay

| Công cụ | Lấy ở đâu |
|---------|----------|
| Source tag | Admin gửi trong group |
| Tracking link | Admin gửi sau khi profile pass |
| Rule + Post-log format | Topic "Nguoi moi" trong group Telegram |

### SOP — Làm Từng Bước

```
Bước 1: Đọc kỹ "3 Quy Tắc Vàng" ở trên (30 giây)

Bước 2: Vào Telegram group → tìm topic "Nguoi moi"
        → Đọc 2 tin nhắn được ghim (pinned)
        → Đây là rule + post-log format

Bước 3: Kiểm tra xem admin đã gửi source tag cho bạn chưa
        → Source tag có dạng: TAG??? (ví dụ: TAG001)
        → Nếu chưa có: nhắn "cho em xin source tag" vào group
        → Đây là MÃ ĐỊNH DANH của bạn — dùng để track người join từ bạn

Bước 4: Nhận tracking link từ admin
        → Link có dạng: https://t.me/+XXXX hoặc https://hermes.bot/join/TAG???
        → Link này sẽ đặt trong X bio của bạn (làm ở Bước 5)

Bước 5: Đặt tracking link vào X bio
        → Mở X (Twitter) → Edit profile → Bio
        → Dán link tracking vào bio
        → KHÔNG để link trong body bài đăng (quy tắc vàng #3)

Bước 6: Nhắn "done" vào group
        → Đơn giản: "em đã đọc rule + đặt link bio xong"
```

### Tại Sao Những Bước Này Quan Trọng?

```
- Source tag = cách hệ thống biết người join từ ai → để tính tiền cho bạn.
- Tracking link trong bio = người đọc click vào profile bạn → thấy link → join.
  Nếu không có link, người ta có muốn join cũng không biết đường.
- Đọc rule TRƯỚC KHI LÀM = tránh bị loại vì vi phạm mà không biết.
- "Done" = admin biết bạn đã sẵn sàng để đi tiếp.
```

### Kết Quả Sau Day 0

```
✅ Bạn có source tag trong tay
✅ Bạn có tracking link trong X bio
✅ Bạn đã đọc rule + hiểu 3 quy tắc vàng
✅ Bạn đã biết ai để hỏi khi kẹt
```

### Nếu Kẹt

```
- "Tôi không biết X bio ở đâu?" → Mở app X → bấm vào avatar → Edit profile → Bio
- "Tôi chưa có source tag?" → Nhắn group: "cho em xin source tag"
- "Tôi không hiểu rule?" → Tag @hiephoang47 trong group: "rule cho vendor mới là gì"
```
